| 参数           | 常用范围                                | 妈妈当前（第2次）          | 所处位置                    |

> 数据来源：多个临床文献及医学平台（博禾医生、复禾健康、Frontiers in Neuroscience等），常用设置为频率130-185Hz、脉宽60-120μs、电压1-3.5V

### 参数说明

- **频率**：130Hz左右是最经典设置，高频刺激用于缓解运动症状；低频(60-80Hz)有时用于改善步态冻结
- **脉宽**：多数患者初始用60μs，电流调高后效果仍不够时，医生会考虑加大脉宽；脉宽越大刺激范围越广，效果更强但副作用风险也增加
- **电流**：从低起步逐步上调，是调参中最常调整的参数

### 后续可调整方向

1. 继续上调电流（目前还有约1~1.5mA空间）
2. 增大脉宽（从60μs到90μs或120μs）
3. 电流 + 脉宽组合调整

> 💡 **为什么医生建议减次数，而不是先减量再看减次数？**
>
> 1. **他当时还没看到异动数据**。调完参那一刻妈妈状态还可以，他判断"DBS更强了→总量可以少一点→减一顿最简单"，这在当时的信息下是合理的。
> 2. **减次数是DBS术后减药的常规做法**：操作简单（少吃一顿就行）、老人不容易搞混、临床验证也多。"先减量再看减次数"需要更细致的观察和调整，不是主流路径。
> 3. **他低估了3/4片在新参数下的峰值**。两根电极同时大幅上调（电极2涨47%、电极9涨80%），3/4片的峰值刚好超线了。但每个病人的阈值不同，没法事先精确计算，只能靠实际反应验证。
> 4. **顶尖医生的经验是基于大样本的**："大多数病人调完参减一顿就够了"。但妈妈对左旋多巴峰值特别敏感，属于个体差异，只有今天实际吃了药才暴露出来。
> 5. **结论：不是医生判断失误，而是3月18日的异动数据是新信息。** 把这些数据发给他后，他大概率会认可"先减量"的思路——因为现在他有了我们给的数据，能做更精准的判断。

#### 减药路线图与理想目标（基于妈妈对美多芭高度敏感的特点）

1/2片峰值已经亢奋贴着异动线，说明1/2片对妈妈来说不是"低剂量"而是**接近上限**。

**预估减药路径**：

| 阶段                        | 方案                     | 美多芭总量/天     | 触发条件             |
| --------------------------- | ------------------------ | ----------------- | -------------------- |
| 调参前                      | 3/4片×4次               | 3片               | —                   |
| **现在**（3月19日起） | **1/2片×4次**     | **2片**     | 3/18两次异动→减量   |

| 1-2月后（预估）             | 1/2片×2次 或 1/4片×3次 | 0.75~1片          | DBS慢性效应充分显现  |
| **理想终态**          | **1/4~1/2片×2次** | **0.5~1片** | DBS+闭环充分优化后   |

**减到3次的判断逻辑（两周后）**：

减次数只影响脸和舌头（空窗期更长），不影响手脚（DBS兜着）。所以：

| 两周后的状态             | 能不能减到3次         | 原因                                                                           |
| ------------------------ | --------------------- | ------------------------------------------------------------------------------ |
| 手脚好、脸也好           | ✅ 放心减             | 全面富余                                                                       |
| **手脚好、脸稍差** | **✅ 也可以减** | 减次数只让脸舌多差30-60分钟，手脚不受影响；而且DBS继续上来后不减反而有异动风险 |
| 手脚差                   | ❌ 暂缓               | 说明DBS还没撑够，不宜拉长空窗                                                  |

> 脸舌的问题**不靠增加药物次数**来解决——调参可以稍微改善但天花板不高（详见「脸和舌头问题的完整分析」章节），脸舌本质上是DBS的结构性短板。但这不是当前最需要担心的问题。
> 而且如果两周后DBS继续增强，不减次数反而可能出问题：DBS更强 + 同样的药量 → 峰值更高 → 3/18的亢奋可能升级成异动。到时候**减次数不是"可不可以"，而是"该减了"**。

**为什么不能完全停药**：

| DBS能管的                | DBS管不太好的（仍需少量药物）    |
| ------------------------ | -------------------------------- |
| 四肢震颤、僵硬、运动迟缓 | 脸部表情、舌头、吞咽             |
| 行走能力                 | 说话清晰度                       |
| —                       | 非运动症状（便秘、情绪、睡眠等） |

> 妈妈已经验证了这一点——药效退了手脚没事，脸和舌头先变差。这些DBS覆盖不到的部分仍需少量药物维持。

**其他药物的远期目标**：

- **息宁**：从1片/晚 → 尝试半片/晚或停掉（等美多芭稳定后再动）
- **普拉克索**：DBS够强后可减量或停掉（优先级最后）

#### 先减量 vs 先减次数——为什么选了减量

起点：调参前 **3/4片 × 4次 = 3片/天**

|          | **先减量（每次减1/4片）** | **先减次数（4次→3次）** |
| -------- | ------------------------------- | ------------------------------ |
| 操作     | 1/2片 × 4次 = 2片/天           | 3/4片 × 3次 = 2.25片/天       |
| 总量降幅 | 减1片（降33%）                  | 减0.75片（降25%）              |
| 单次峰值 | **降了**（3/4→1/2）      | **没变**（还是3/4片）    |
| 覆盖次数 | 不变，4次                       | 少1次，空窗期更长              |

**减量的优势**：

- ✅ 直接压峰值，针对异动最对症
- ✅ 覆盖次数不变，空窗期不会变长，脸和舌头的覆盖时间不受影响

**减量的劣势**：

- ❌ 总量降得更多（33%），万一DBS兜不住可能药效不够
- ❌ 每顿都要改剂量，掰药麻烦一点

**减次数的优势**：

- ✅ 操作简单，少吃一顿就行
- ✅ 总量降得少一些（25%），更保守

**减次数的劣势**：

- ❌ 峰值没降，每次还是3/4片，异动可能照样出现
- ❌ 空窗期变长（3.5小时→4-5小时），脸和舌头在空窗期会更早变差

**核心区别**：

> - **减量**解决的是"每次吃多了"（峰值过高→异动）
> - **减次数**解决的是"总共吃多了"（总量偏高→可以少一顿）

**为什么这次选了减量**：3月18日的异动说明问题出在峰值——当天已经少吃了一次（11:00跳过），等于已经"减次数"了，但下午吃3/4片照样异动。说明3/4片的峰值在新参数下过线了，减次数解决不了峰值问题。

**什么情况下该选减次数**：如果没有异动，只是医生觉得"DBS强了，药可以少吃点"，那减次数就是对的——简单、好操作、降幅温和。但异动出来后，优先级变了：先把峰值压下来，再谈减次数。

#### 还需要等慢性效应出来吗？（3月25日讨论）

今天距3月18日调参才**1周**，慢性效应理论上在1-4周内逐渐显现，确实还没完全出来。

**但关键问题是：慢性效应能帮舌头吗？**

**大概率帮不了。** 原因：

- 慢性效应的本质是DBS持续刺激下大脑神经网络的重塑
- 但神经网络再怎么重塑，也是在**电极能覆盖的范围内**重塑
- 电极覆盖的就是管手脚的那片区域（STN运动区主体），脸和舌头的控制区在边缘和更远的位置
- 慢性效应不太可能把"够不到"的地方变成"够得到"

**慢性效应真正能改善的**：早上起床时手脚从"稍微慢一点"→"不慢了"，也就是DBS裸奔兜底成绩从80分提到85-90分。但这跟舌头有效覆盖时间没有关系。

**所以：不管再等1周还是3周，舌头的覆盖时间大概率不会因为慢性效应明显变长。**

**舌头要变好只有两条路**：

1. **调参**——换触点、加脉宽，把刺激范围往脸舌方向推
2. **药物覆盖**——但现在已经是4次/天，间隔3.5小时，再加次数不现实

## 关于调参后效果的正确理解

### 第1次开机 vs 第2次调参：为什么这次不会"几天后又不行"

|                        | 第1次开机（2月4日）                 | 第2次调参（3月18日）           |
| ---------------------- | ----------------------------------- | ------------------------------ |
| **性质**         | 首次开机，全新刺激                  | 在已适应的基础上提高参数       |
| **"蜜月期"效应** | ✅ 有，效果被放大                   | ❌ 已过，不会再有              |
| **微损毁效应**   | ✅ 有（手术植入电极本身的临时改善） | ❌ 已消退完毕                  |
| **效果消退风险** | 蜜月期+微损毁效应消退后感觉"变差"   | 无虚假加成，看到的就是真实效果 |

- 第1次开机后"几天就不行了"，是因为**蜜月期效应 + 微损毁效应**消退，并非DBS失效
- 第2次调参后**不存在这两个临时加成**，今天观察到的效果（包括异动）就是DBS的真实作用
- 出现异动恰恰说明DBS参数调高后**确实在有效工作**
- 结论：**第一次是从"虚高"往下掉，这次是从"真实起点"往上走**

| **2-4周**   | 效果稳定甚至继续改善（DBS的"慢性效应"逐渐显现） |

### 短期（每次调参后2-4周内）

- **临床研究的普遍结论**：每次调参后，效果会在2-4周内逐渐显现和稳定
- DBS存在"慢性效应"，部分症状（僵硬、步态等）不会调完立刻见效，大脑需要时间适应新参数
- 所以刚调完感觉不明显是正常的，不要急于下结论，耐心观察日常功能的实际变化

### 为什么调参后效果还会继续变好？——DBS的"慢性效应"

DBS的效果分两部分：

|                      | 急性效应（立刻见效） | 慢性效应（逐渐显现）         |
| -------------------- | -------------------- | ---------------------------- |
| **起效时间**   | 调参后几秒到几分钟   | 数天到数周                   |
| **作用机制**   | 直接抑制异常神经放电 | 神经网络重塑、突触可塑性改变 |
| **改善的症状** | 震颤（最快）         | 僵硬、步态、姿势稳定性等     |

慢性效应的原理：

1. **神经网络重塑**：电刺激不只是"压住"异常信号，还会促进大脑运动回路重新建立更正常的工作模式
2. **突触可塑性**：持续的电刺激改变神经元之间的连接强度，让运动控制通路变得更顺畅
3. **下游脑区适应**：刺激目标核团连接着很多脑区，这些下游脑区也需要时间适应上游信号的变化

> 比喻：急性效应像把堵住的水管疏通了，水立刻能流；慢性效应像河道疏通后，周围的土壤、植被慢慢适应新水流，整个生态逐渐改善

### 慢性效应的客观证据（不是主观感觉，是仪器测出来的）

| 证据类型               | 具体发现                                                                                                                        |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| **PET脑扫描**    | DBS开机数周后，运动皮层代谢活动逐渐正常化——脑扫描仪上能直接看到的物理变化                                                     |
| **电生理记录**   | 持续刺激后，STN的异常振荡（beta波）逐渐减弱——神经放电模式在客观改变                                                           |
| **临床量表评分** | 大样本研究显示UPDRS运动评分在调参后1-4周持续改善，不是第一天最好然后掉                                                          |
| **停机实验**     | DBS用几个月后关机，效果不是立刻消失，而是**慢慢退**（有的症状几天后才回来）——说明大脑结构已经被改变，不只是"通电才有效" |

### 为什么日常感觉不到慢性效应在起作用

**因为容易看错指标：**

| 容易盯着看的（看了也看不出来） | 慢性效应真正改变的                                 |
| ------------------------------ | -------------------------------------------------- |
| 脸舌好不好                     | ❌ 脸舌不是DBS强项，慢性效应在脸舌上的改善很微小   |
| 每天对比差别大不大             | ❌ 每天看就像每天看小孩，觉得没长高                |
| 白天有药时的状态               | ❌ 药效把DBS的改善"淹没"了，90分→95分日常分不出来 |

**正确的观察方法：看早上没药时的状态（纯DBS兜底）**

| 时间点            | 现在（调参后第4天） | 预期2周后（如果慢性效应起作用）              |
| ----------------- | ------------------- | -------------------------------------------- |
| 早上起床（纯DBS） | 手脚稍慢，脸舌差    | 手脚不再慢（这就是慢性效应），脸舌可能还是差 |
| 白天有药          | 全好                | 全好（看不出区别）                           |

> 📌 **验证计划：3月28日（调参后第10天）早上起床状态 vs 3月22日早上起床状态**
>
> - 如果手脚不再"稍慢"了 → 慢性效应实锤
> - 如果脸舌还是差 → 正常，DBS对脸舌天花板就在那里
> - 如果一切都没变 → 再重新评估

### 长期趋势：坚持调参是关键

- 当前DBS调整空间**大约只用了20-30%**——电流还有一半多余量，脉宽和频率完全没动，8个触点才用了2个，闭环功能更是未来的大杀器，远远没到"调不动"的阶段
- 前6个月是调参黄金期，大脑可塑性最强，调参效率最高
- 多数患者在术后6-12个月达到明显且稳定的改善

### 约3个月后效果可能不足——原因和应对

**为什么**：大脑会逐渐"习惯"电刺激（神经适应/耐受），类似于药物的耐受性

| 阶段              | 大脑反应                                   |
| ----------------- | ------------------------------------------ |
| 调参后头几天~几周 | 新参数是"新鲜刺激"，反应灵敏，效果明显     |
| 1-3个月           | 大脑逐渐适应，神经回路开始"重新平衡"       |
| ~3个月            | 适应基本完成，同样的参数产生的效果比最初弱 |

**具体机制**：

1. **突触可塑性**——被持续刺激的神经元会下调对电信号的响应敏感度
2. **神经回路代偿**——大脑发展出绕过刺激区域的替代通路，部分抵消DBS效果
3. **电极周围组织变化**——电极植入处可能逐渐形成微小胶质瘢痕，增加电阻

**应对策略：优先调参，不是加药**

|          | 调参（调DBS）                     | 加药                               |
| -------- | --------------------------------- | ---------------------------------- |
| 副作用   | 几乎没有，调不好可以改回来        | 加药→异动风险增加，妈妈又特别敏感 |
| 可逆性   | 完全可逆，参数随时调              | 药物加上去容易、减下来难           |
| 储备空间 | 6个触点没用、电压/脉宽/频率都能调 | 药量在减，加回去是走回头路         |
| 长期趋势 | DBS越调越精准                     | 药越吃越多是最想避免的恶性循环     |

> **调参 → 换触点组合 → 调电压/脉宽/频率 → 实在不行再考虑药物微调**
> 药物只是最后的兜底手段。这也是为什么现在趁DBS效果好要尽量减药——给未来留足空间。

## 脸和舌头问题的完整分析

### 为什么DBS对脸舌效果有限——解剖结构决定的

| 身体部位       | 在大脑运动区的位置                                     | DBS电极能不能够到                                 |
| -------------- | ------------------------------------------------------ | ------------------------------------------------- |
| 手、脚、四肢   | STN（丘脑底核）中的**运动区主体部分**            | ✅ 正好在电极刺激范围内                           |
| 脸、舌头、吞咽 | STN的**腹内侧边缘** + 更靠下的**脑干区域** | ⚠️ 边缘位置能覆盖一部分但不完整；脑干完全够不到 |

> 比喻：电极是一个手电筒，手脚在光圈正中间，脸舌一部分在光圈边缘、一部分在光圈外面。可以调角度、换灯泡让光圈大一点——边缘能亮一些，但光圈外面怎么调也照不到。

### 美敦力PRC能把脸舌调到多好？

PRC是目前最顶级的设备，但"最顶级"体现在**精度和安全性**上，不是说它能突破解剖结构的限制。

| PRC的优势                      | 对脸舌的意义                                                                 |
| ------------------------------ | ---------------------------------------------------------------------------- |
| **BrainSense脑电感知**   | 调参时有实时脑电数据参考，能更精准找到"对脸舌有帮助但不产生副作用"的刺激区间 |
| **精细电流塑形**         | 可以更精确地把刺激范围往脸舌控制区方向"推"一点，减少对不该刺激区域的扩散     |
| **闭环自适应（等审批）** | 药效低谷时自动把DBS拉到最大，在天花板范围内尽量多覆盖一些                    |

2024年Brain Stimulation期刊研究发现：通过精确定位STN内的不同亚区，**右侧STN的特定位置刺激可以改善言语和声音质量**，且低到中等强度比高强度效果更好——说明"对的位置 + 对的强度"确实能帮到言语。

### 换触点/调参数实际能改善多少？

| 操作                                     | 能做到什么                                 | 做不到什么                            |
| ---------------------------------------- | ------------------------------------------ | ------------------------------------- |
| **换触点**（激活靠近腹内侧的触点） | 把刺激范围稍微往脸舌控制区偏移，多覆盖一点 | 偏移太多可能影响手脚效果或产生副作用  |
| **加大脉宽**（60→90μs）          | 扩大刺激范围，可能多覆盖边缘区域           | 范围太大可能刺激到不该刺激的地方      |
| **多触点组合**                     | 更精细地塑造刺激范围的形状                 | 本质上还是在STN附近做文章，够不到脑干 |

> **实际能做到的**：
>
> - 药效好的时候脸舌也好 → ✅ 现在就是这样
> - 药效尾段脸舌"稍差"变成"差得更少" → ✅ 有希望，靠精细调参+闭环
> - 药效退了脸舌也完全没感觉 → ❌ 做不到，哪个设备都做不到

### 闭环对脸舌的加成（ADAPT-PD研究数据）

2025年9月JAMA Neurology发表的ADAPT-PD研究（美敦力自适应DBS大型临床试验，53名患者）：

| 指标                         | 数据                                         |
| ---------------------------- | -------------------------------------------- |
| 闭环模式下长期构音障碍发生率 | **0%**（长期随访中无人报告言语副作用） |
| 闭环 vs 固定模式运动控制     | 91%的患者维持了同等水平                      |
| 节能                         | 比固定刺激**省电13-16%**               |

闭环对脸舌的帮助逻辑：药效低谷时DBS自动加强刺激→在天花板范围内拉满→脸舌"差的那一阵"会缩短、程度减轻。

### 干细胞治疗帕金森——可能是未来最大的变量

#### 先说结论：干细胞能治愈帕金森吗？

> **不能治愈，但是"大幅充值"——一次性补充大量多巴胺神经元，多争取很多年的好状态。**

帕金森有**两层问题**：

| 层次                                                                | 问题                                                   | 干细胞能解决吗 |
| ------------------------------------------------------------------- | ------------------------------------------------------ | -------------- |
| **表层**：多巴胺神经元死了，多巴胺不够 → 抖、僵、慢          | ✅**能**——移植新的多巴胺神经元，直接补上       |                |

> 比喻：帕金森像一个**不断漏水的水池**：
>
> - 药物 = 不断往里加水（补充多巴胺），但不堵漏洞

> - **干细胞 = 往池子里大量灌水（补充大批新的产水细胞）**——水位大幅升高，但**漏洞还在**，新补的水最终也会慢慢漏
> - **真正的治愈** = 堵住漏洞（阻止α-突触核蛋白杀死神经元）——这个目前还没做到

## DBS闭环功能——未来最大的升级

### 什么是闭环

- **开环（现在）**= 空调只能设一个固定温度——白天太热它不会加强，晚上太冷它不会升温
- **闭环（未来）**= 智能空调，有温度传感器，**随时感知、随时调节**，全天舒适

开环模式下DBS是"定速巡航"——不管药效高峰还是低谷，刺激强度都一样：

```
药效高峰：  ████████████  ← DBS不知道，继续同样强度 → 叠加过量→容易异动
正常状态：  ████████      ← 刚好
药效低谷：  ███           ← DBS不知道，还是同样强度 → 不够→脸舌变差
```

闭环模式下DBS根据脑电信号实时调节：

```
药效高峰：  ████████████  ← DBS感知到→自动降低刺激 → 刚好，不异动
正常状态：  ████████      ← DBS感知到→维持 → 刚好
药效低谷：  ███           ← DBS感知到→自动加强刺激 → 补上来，脸舌也覆盖
```

### 闭环的主要作用（按重要性排序）

| 排序              | 作用                   | 说明                                                               |
| ----------------- | ---------------------- | ------------------------------------------------------------------ |
| **🥇 最大** | **消除状态波动** | 全天状态平稳，不再忽好忽坏。这对生活质量的提升是**质变级别** |
| 🥈                | **自动防异动**   | 药效高峰时自动降低DBS刺激，不用靠精确掐药量来避免异动              |
| 🥉                | **有利于减药**   | 药效低谷时DBS自动加码补上，减药后也不怕撑不住                      |
| 4                 | **减少调参频率** | 机器自己在实时"微调"，不用频繁跑医院调参                           |

> 闭环最大的价值不是"少吃几片药"，而是**不用再数着时间等药效、担心药退了变差、担心峰值异动**——机器帮你管着，状态全天平稳。

### 对妈妈减药的意义

开环下减药是**走钢丝**——峰值不能过线（异动），低谷又不能太差（脸舌），只能在中间找窄窄的安全区间。

闭环下减药是**走大路**——峰值高了DBS自动退让，低谷来了DBS自动加码，两头都有自动调节，减药的顾虑大大减少。

> 等国内开放闭环功能后，妈妈的减药空间会再大一截，理想终态（0.5片/天甚至更低）更有可能达到。
